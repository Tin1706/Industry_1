Can interface protection only be used for medium voltage connections or also for low voltage connections?

There are two versions of the interface protection: one with a nominal voltage of 100V for an indirect medium voltage connection, the other with a nominal voltage of 400V for a direct low voltage connection. The CEI0-16 standard requires the use of a protection relay such as the NV10P only for medium voltage connections. In the case of low-voltage connections, the NV021 relay must be used.

DATA-LOGGER function required by Technical Italian Standard CEI-016: What is it and when is it needed?

The DATA-LOGGER function is an optional function present in the protective relays NA016, NA10, NA30 and NA60. The italian technical Standard CEI-016 requires it to be provided in the General Protection (PG) only if the General Device DG (Circuit Breaker controlled by the protective relay PG) is equipped with a current-carrying trip coil. If, on the other hand, the General Device DG is equipped with a undervoltage trip coil, the DATA-LOGGER function is not required.

Are Thytronic relays used as PG CEI 0-16 always complete with data logger? 

The presence of the data logger depends on the specifications of the relay and the type of control used. If the relay operates a switch release command by means of a minimum voltage coil, the use of the data logger is not necessary. However, if a current-throwing command is used, tracking on the data logger is necessary.
Most CEI 0-16 certified Thytronic relay models are already equipped with a data logger. However, for the NA60 relay alone, it is necessary to purchase an additional module called MRI. For the NA016, however, the presence or absence of the data logger is determined by the purchase code: NA016....5M100 with logger, 5M000 without logger.

NV10P factory parameters for interface protection CEI 016: check them before use!

The NV10P relay is an interface protection device conforming to the italian standard CEI 0-16. In order to facilitate use of the device, the relay is delivered with a pre-programmed default configuration, including the threshold and delay values required to achieve compliance with the standard. However, it is important to note that the user must check that the factory configuration is suitable for the application and modify it if necessary. It should be noted that the adjustments only concern parameters that are essential for CEI 0-16 compliance, while all other functions available on the relay remain unchanged from the default factory configuration. To further facilitate the use of the NV10P relay, ThyVisor allows you to insert a configuration file generated with the calibration-PI  interface directly into the device. Check the factory configuration of your NV10P before use to ensure maximum efficiency and safety of the device.

Does the NV10P interface protection perform the automatic reclosing operation? 

The NV10P interface protection performs the automatic reclosure operation but only by means of the optional reclosure management software (NV10P-arf-0206).

How to find the SW version of a protective relay?

There are several options for accessing this information.
  1. The nameplate data can be found on the Test Certificate included in the device's packaging.
  2. Alternatively, the READ / INFO menu can be accessed using the MMI interface (display + keyboard) with the device powered.
  3. Finally, the licence plate data are also available on the configuration file editable with the ThyVisor software

What is the typical calibration of protective relays? 

Our protective relays leave the factory without a specific configuration and typical calibration with the exception of the NV10P protective relay. There is no generally valid set of values. Each installation requires a relay configuration according to the peculiarities of the installation. Only after a relay calibration study can the parameters to be entered in the protective relay be defined

What are the relay's default parameters? 

Thytronic protection relays are supplied without any parameter settings. You need to configure them using the ThyVisor software, which can be downloaded free of charge from our website.  

Once you have downloaded the software you need to connect to the protective relay via USB cable for the PRO-N family or via network cable for the XMORE family.
TIPS: do not forget to download also the template and USB driver for a better experience

Is ThyVisor a free software?

Thyvisor is the software for supervising and managing Thytronic protection relays. The ThyVisor base version is free, but you need a license for additional features like oscillography and PLC. Once you have an oscillographic record, you can open it using every third-party Comtrade file viewer; our license is not necessary. Once you have compiled an Instruction Language code, you can download it to every relay without our license.

How can I copy the settings of an old relay in a new one?

To copy the calibrations of an old relay into the same new model: 
  1. open the Thyvisor software, 
  2. go to communication with the relay, 
  3. With the coursor on the relay icon, press the right mouse button and select File > Merge Settings File option. 
  4. Load the old File, the program will automatically convert it

Is PLC functionality present in all Thytronic protection relay models? 

PLC functionality is present in all models of the ProN and XMORE family. The upload of a PLC program already compiled in the protection is free. To create or edit a source file for PLC functionality, the purchase of the optional ThyPLC licence is required.

Is the oscilloperturbography functionality present in all Thytronic protection relay models? 

The oscilloperturbography functionality is present in all models of the ProN and XMORE family. Its configuration is free. The purchase of the optional ThyOSC licence is required to retrieve the file from Thytronic protection.

How to change Thytronic protective relay settings?

To change Thytronic protective relay settings, access the Thyvisor programming software. Click on IED > Open. Select the relay and click on SET. Select the parameters you want to adjust, change the values and click OK to save the changes.

I cannot connect to the relay via the ThyVisor software, what can I do? 

In case you cannot connect to the relay via the Thyvisor software, please follow these steps before contacting our customer service
  1. Check that you are using the latest version of the ThyVisor software available on our website.
  2. Verify that the communication port via front USB connection is correct 
  3. In case of connection via Ethernet port, you need to assign a valid IP address for communication between PC and relay. 
For further support, you can contact us on Whatasapp at +39 0257495722 or write to tecnico@thytronic.it

If I have more than one PC, how many PLC or Oscillographic licenses do I need?

The license is available either as a file or as a USB key, and it's possible to share the USB key on every PC.

How to set/update date and time on Pro-N protection?

To set or update the date and time on Pro-N protection, simply use the ThyVisor software. Connect to the software, click on IED > Procedures > Clock Sync, and make sure you are online with the protection. After doing so, the protection's clock and calendar will be synchronised with those of the computer.

If I have to connect directly to a low-voltage network and I have a nominal voltage other than 400V, how do I adapt the inputs of the interface protection? 

If I have to connect directly to a low-voltage network with a nominal voltage other than 400V, it is necessary to change the UN nominal voltage parameter in the SET / BASIC menu in the Thyvisor software.

How to Prepare an Offline Configuration File: Step-by-Step Instructions

If you need to prepare an offline configuration file "from scratch," don't worry - you can do it without being connected. However, it's important to use the correct template file to achieve the best results. If you're unsure which one to use, we recommend connecting to the relay at least once. This will automatically open the correct template, save it to your PC, and allow you to work on it. Additionally, it's important to stay updated with the latest versions available, which you can find on our website under the Support/Software/Template or Resources/Software section.

I've a setting file on my PC. How can I send to the relay?

To send a setting file from your PC to the relay, make sure the file is compatible to avoid corrupting the internal database (see our previous FAQ). Once you have confirmed compatibility, select the relay icon and follow these commands in ThyVisor: IED ribbon > procedures > Send Data. Click on "Start" to initiate the process.

How can I download on my PC the setting of a Thytronic Protection Relay?

You can download on your PC the setting of Thytronic protection relay using Thyvisor software- The correct way to get data stored in the memory of the relay is the command in ThyVisor Procedures -> Receive Data.
N.B Do not use the File -> Save (or Save As) as it is only for network saving

Can I setup a Protective Relay Network?

Yes, our relays make it easy to set up a protective network. Simply assign an address to each relay and follow standard PC network instructions.

How can I program a relay as a PLC?

You can send your compiled instruction language code to the relay. The PLC compiler is working under license. Please write us at thytronic@thytronic.it for more information.

How can I programm a ModBus communication?

Use our programming manual. Each manual is related to the firmware version. Write to us at thytronic@thytronic.it indicating model, code, FW version and we will send you the correct manual according to the relay version you have.

How can I avoid shielding interference problems resulting from installations in environments with strong electromagnetic disturbances and/or with long distances? 

To avoid electrical interference in environments with electromagnetic noise or long distances, the best option is to use fiber optic connections or well-shielded cables. These methods offer greater protection against electrical interference than traditional wired connections. In addition, they can provide higher connection speed and reliability.

What cable does the RS485 serial port require? 

The RS485 serial port requires a twisted and shielded cable

What is the difference between the Ethernet port and the RS485 port on the relay?

You have two communication ports to connect your devices to the relay: 
  • an Ethernet port identified by D1 
  • an RJ45 port identified by E1
The Ethernet port is the main connection port for Ethernet-compatible devices, while the lower RS485 port can be used to connect to auxiliary modules such as MID16, MRI, MCI, and MPT, which use the proprietary ThyBus protocol developed by Thytronic. To connect an auxiliary module to the relay, simply connect the module to the RS485 port with the supplied network cable and follow the instructions in the product manual.

 Are the relay output contacts polarized? 

No, the relay output contacts are voltage free and can be programmed as maintained/not maintained, energized/not energized, all separately.

Are the input binary terminals to be polarized? 

Input binary terminals should be polarized with a range of 24 to 230 Vac/dc

Can I use any output contacts to build a system in accelerated logic? 

There are dedicated ports in our relays for managing logic selectivity among various Thytronic devices. We highly recommend using the dedicated output for continuous check and, if possibile, to use fiber optic in high disturbance environments.
